Linear algebra and numeric library

Puzlet supports an extensive numeric library, based on the [http://numericjs.com, numericjs] module:
  • Linear algebra
    • linear equation solution
    • matrix inverse
    • determinant
    • eigenvalues and eigenvectors
    • LU decomposition
    • singular value decomposition (SVD)
    • norms
  • Sparse matrices
  • Coordinate matrices and partial differential equations (PDEs)
  • Ordinary differential equations (ODEs)
  • Linear/quadratic programming and optimization
  • Cubic splines
  • FFTs
