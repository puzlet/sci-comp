Puzlet for Scientific Computing

With Puzlet, you can do scientific computing directly in your browser:
  • [./func, Math functions]
  • [./vect, Vectors and matrices]
  • [./complex, Complex numbers]
  • [./la-num, Linear algebra and numeric library]
    • linear equation solution, inverse, determinant
    • norms, eigenvalues, eigenvectors
    • LU decomposition, singular value decomposition (SVD)
    • sparse matrices, coordinate matrices, PDEs, ODEs
    • cubic splines, linear/quadratic programming, optimization
    • FFTs
  • [./utils, Printing, plotting, and utilities]
Puzlet's language is a fusion of [http://coffeescript.org, CoffeeScript], [http://numericjs.com, numericjs], [http://flotcharts.org, flot], and [http://paperjs.org/reference/paperscript, PaperScript] operator overloading (e.g., +, -, *, / for vectors, matrices, and complex numbers). This page is an introduction and a quick reference guide. The links above show detailed reference/syntax examples. You can edit and run any of the code below (shift+enter to run).
